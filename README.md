# [dprint][d] for [Nova][n]

See [the extension README](./dprint.novaextension/README.md) for details. This host package only exists to make *building the extension* easier.

[d]: https://dprint.dev
[n]: https://nova.app

## Development

Install dependencies:

```sh
pnpm install
```

Build the package:

```sh
pnpm build
```
